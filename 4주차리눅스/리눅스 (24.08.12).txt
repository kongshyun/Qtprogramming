[복사하기]

		-> a 내용을 ac 로 복사

		hello

[확인하기 i 노드] -> a와 al은 번호가 같음
	iam@boy:~$ ls -i
	524299 a  524300 ac  524299 al  524301 b  524302 c

	-> 파일의 실체 : i 노드

_
[링크]
1. hard link
	ln a la
	-> 파일에만 링크를 걸수 있음.( link )

2. soft link ( symbolic )
	ln -s a as
	->as도 a와 값이 같음. BUT i 노드가 다르다!! 다른파일이다.

	-> ls- l 해보면, " as -> a "
	-> 파일과 디렉토리에 링크를 걸수 있음.(link / dirs)
